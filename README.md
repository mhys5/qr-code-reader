# qr-code-reader
画像からQRコードを読み取りURLを表示する静的Webアプリ。
背景が透過されている画像や周囲に余白のないQRコード、カラーQRコードも読み取り可能。

## 使ってみる
https://mhys5.github.io/qr-code-reader/

## アップロード対応拡張子
PNG / JPG（JPEG）/ GIF / WebP / BMP / SVG

ブラウザが画像として表示できる形式に対応。
TIFFやHEICは、多くのブラウザがデコードできないため非対応。

## 読み取り履歴について
履歴を最大10件までlocalStorageに保存。
また、各履歴のメモ欄には自由にテキストを入力可能。
履歴を削除したい場合は、各行のゴミ箱アイコンをクリックするか「すべて消去」ボタンを押下することで削除可能。
